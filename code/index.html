Game Instructions

This goal of this game is to guess who the secret character is. At the start of each game there is a secret person randomly choosen by the computer.

Your objective is to try and guess who it could be. Either use the dropdown meny to the left by choosing a character traite. You can then press the Find out button

to filter out characters that either have or dont have the attribute that you choose. When ever you feel ready you can click the guess button on each card to guess on

that specific character. If you need these intructions again there will be a button to the left to bring you right back here.
